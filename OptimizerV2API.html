Replanapicontroller

Up 
post /replan
Generates a Planning Solution for a given Next Release Problem (replanUsingPOST)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body NextReleaseProblem (required)
Body Parameter — The Next Release Problem to replan

Return type

PlanningSolution

Example data

Content-Type: application/json
